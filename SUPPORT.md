# 支持 AgentDock

感谢你有兴趣支持 AgentDock 的发展!🙏

## ⭐ 给项目点 Star

这是最简单、也是最有帮助的支持方式:

👉 [**在 GitHub 上给 AgentDock 点 Star**](https://github.com/ACCSCI/AgentDock)

Star 数帮助我们:
- 让更多人发现项目
- 提升项目可信度
- 激励维护者持续投入

## 🐛 报告 Bug、💡 提功能建议

- [Bug Report](https://github.com/ACCSCI/AgentDock/issues/new?template=bug_report.md)
- [Feature Request](https://github.com/ACCSCI/AgentDock/issues/new?template=feature_request.md)

## 💻 贡献代码

参考 [CONTRIBUTING.md](CONTRIBUTING.md)。

- `good first issue` 标签 — 适合首次贡献
- `help wanted` 标签 — 需要社区帮助

## 📖 改进文档

发现文档错误或不清晰的地方?提交 PR 或 Issue:
- [Documentation Issue](https://github.com/ACCSCI/AgentDock/issues/new?template=documentation.md)

## 📢 分享项目

- 写博客 / 录视频介绍 AgentDock
- 在社交媒体(X、微博、知乎)上分享
- 在你的项目 README 中推荐 AgentDock

## 💰 资金支持

目前 AgentDock 还没有正式的赞助渠道。如果你希望资金支持项目发展:

- 请先联系维护者讨论资金用途
- 所有赞助将用于:服务器、域名、CI 资源,以及给贡献者的奖励

## 🔒 报告安全漏洞

请**私下**报告,详见 [SECURITY.md](SECURITY.md)。
